<내부함수>
내부함수란? 함수내에 선언된 함수이다.  내부함수는 선언된 함수내에서만 사용할 수 있다.
내부함수를 사용하는 이유는 충돌을 피하고자 사용한다.

ex) A란 개발자가 만든 함수와 B란 개발자가 만든 함수를 가져와 사용할때 내가 만든함수와
	용도는 다르지만 이름이 같을때는 한 함수밖에 사용하지 못한다.
	그래서 그 함수에서만 필요한 함수는 함수안에 만들어서 충돌을 피하고자 사용한다.
-사용방법-
function ex1(a,b) {
  // ex1이란 함수 안에서 다시 
	function ex2(c) {
       //   함수를 선언해서 사용한다.
	}
}

<자기호출함수>
자기호출함수도 다른 개발자에게 영향을 주지 않게 함수를 생성하자마자 호출하는 경우이다.
내부함수와 마찬가지로 다른함수와 충돌하지 않으려 사용한다.

-사용방법-
(function() { alert('자기호출함수'); } ) () ; 이렇게 사용한다.
일반함수를 사용할때는 함수이름(); 이렇게 사용하는것처럼
자기호출함수를 사용할때도 ( ) ( ); 인데 첫번째 () 안에 함수를 정의해주는것이다.
이 함수는 일회성이다.
