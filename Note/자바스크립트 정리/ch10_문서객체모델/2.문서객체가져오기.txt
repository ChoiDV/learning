<문서 객체 가져오기>
문서 객체를 id를 이용해서 가져오기

1. getElementById(id)를 이용해서 가져오기 
위의 태그의 id 속성이 id와 일치하는 문서 객체를 가져 온다.
이 방식을 사용하려면 body에서 가져올 태그들에 id를 넣어주어야 가져올 수 있다.

2. getElementsByClassName('class명')을 이용해서 가져오기
위의 태그의 매개변수의 class명과 일치하는 class들을 가져 온다.
이 방식을 사용하려면 body에서 가져올 태그들에 class들을 넣어주어야 가져올 수 있으며,
중복된 class명을 사용할 수 있기 때문에 객체를 가져올때 배열로 받는다.

가져올 클래스가 1개이면 배열0번방에 2개이면 0,1번방에 차례차례 들어온다.

3. getElementsByName(name);
태그의 name 속성이 name과 일치하는 문서 객체를 가져온다.
이 방식 또한 중복된 name을 가질수 있기 떄문에
객체를 가져올 때 배열로 받는다.

4. getElementsByTagName('<태그>');
'<태그>' 인 객체들을 가져온다.
이 방식도 여러개의 태그가 있을수 있기때문에 객체를 가져올 때
배열로 받는다.
