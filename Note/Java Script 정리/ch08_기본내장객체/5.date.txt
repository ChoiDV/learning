<Date>
Date함수는 현재의 시간 날짜 요일 월 연도 등 시간을 알려준다.
new Date(); 로 생성할 수 있다.

new Date()의 내장함수에는 

.toLocaleString() : 연도4자리.월.일.  오전/오후  시:분:초  의 형식으로 알려준다.

.toLocaleDateString() : 연도4자리.월.일 의 날짜 형식으로 알려준다.

.toLocaleTimeString() : 오전/오후 시:분:초 의 시간 형식으로 알려준다.


new Date(); 를 사용할때 매개변수로 특정 시점을 넣어줄 수 있다.

var day = new Date(2022,7,18,15,20,0); 은
2022년도 8월 18일 오후 3시 20분 0 초 이다.
컴퓨터에서 달은 0부터 시작이기때문에 원하는 달의 -1 로 해서 넣어주어야한다.
