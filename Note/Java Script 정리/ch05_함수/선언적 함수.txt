<선언적 함수>
선언적 함수는 익명함수와 다르게 함수의 이름을 같이 선언한다.

-사용법-
function fun( ) { " ? " }; 
? 는 fun함수를 호출했을시 실행할 명령들을 넣어준다.

*선언적 함수와 익명함수와 차이*
: 웹브라우저는 script 태그 내부의 내용을 읽기전에 선언적 함수부터 읽는다.
  그래서 선언적 함수의 위치에 상관없이 함수를 사용할 수 있다.

: 선언적 함수는 선언하고 위나 아래 어디서든 호출할 수 있다.
 
: 선언적 함수는 함수 선언이기 때문에 ; 을 쓰지 않아도 된다.

*함수의 오버로딩*
'자바 스크립트에는 오버로딩이 안된다.'
매개변수 개수로 함수가 나뉘어지지 않는다.
이름이 같은 함수를 매개변수 2개짜리와 3개짜리를 만들면
더 나중에 선언된 함수만 남아있고 먼저 선언된 함수는 사라진다.

*함수 호출 시 매개변수 개수*
매개변수가 더 많을때 : 
	매개변수 2개짜리 함수가 있을때 
	그 함수를 호출할때 매개변수를 3개이상 적으면 자바스크립트에서는 무시해준다.
	즉 앞의 2개를 매개변수로 받고 뒤를 무시해준다.
매개변수가 더 적을때:
	매개변수 2개짜리 함수가 있을때
	그 함수를 호출할때 매개변수를 1개만 적는다면 그 값은 첫번째 매개변수에 들어가고
	두번째 매개변수 값에는 undefined가 들어간다.
	즉 매개변수를 첫번째 부터 넣어주고 없는값에는 undefined를 넣어준다.
