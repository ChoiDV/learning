<조건문>

<if문>
if문은 자바와 사용법이 동일하다.
if ( 조건식) {
 조건식이 참일 경우 실행
} else {
 조건식이 참이 아닐 경우 실행
}

<switch문>
switch문도 자바와 사용법이 동일하다.
switch( 변수) {
   case 값 1 :
	실행문;
	break;
   case 값 2 :
	실행문;
	break;
    default :
	값이 1, 2 가 아닌 그 외의 값들의 실행문;
	break;
} 


<자바 스크립트에서만의 짧은 조건문>

( 숫자 % 2 ==0 ) ||  alert("홀수입니다.");
( 숫자 % 2 ==0) && alert("짝수입니다.");
