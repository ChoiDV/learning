<반복문>
반복문은 자바와 사용법이 동일하다.

자바스크립트에 반복문은 3가지가 있다.

1. 일반 for문 
for( 초기값; 조건식; 증감식;) {
 실행할 명령문
}

2. for-in 문
for( 변수 in 객체 ) {
 객체의 모든 열거할 수 있는 프로퍼티의 개수만큼 반복적으로 실행하고자 하는 실행문;
}

3. .forEach() 함수
forEach 함수는 파라미터로 콜백함수를 받는다.
이 함수는 배열이나 객체에 사용한다.

.forEach( function( 요소를 받을 변수 , 인덱스번호를 받을 변수){
  배열의 인덱스번호의 요소들을 출력하는 실행문 ;
}
