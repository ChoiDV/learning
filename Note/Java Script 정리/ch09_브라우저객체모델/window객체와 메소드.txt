<window 객체와 메소드>
window 객체는 많은 속성과 메소드가 있다.
window 객체는 브라우저 기반 자바 스크립트의 최상위 객체이다.
alert(), prompt(), var 등등 모두 window객체의 메소드이다.

window.사용할 함수 로 window 함수를 사용한다.

자바스크립트의 작동 방식은 위에서부터 차례차례 읽어 들이는 방식이다.
그렇기 때문에 작성위치에 따라서 오작동이 일어날 수 있습니다.
그런 문제를 해결하기 위해 .onload 함수를 사용한다.

window.onload 함수는 웹브라우저 내의 모든 요소가 준비가 되어야 
실행이 되도록 하는 함수이다.
즉 실행할것들을 모두 실행한뒤 실행하라는 뜻이다.
