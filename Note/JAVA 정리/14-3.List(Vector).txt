Vector 란?
Collection 프레임워크의 일부이며
Vector 또한 ArrayList와 동일한 구조를 가지며 배열의 크기가 늘어나고,
줄어듬에 따라서 자동으로 크기가 조절이 된다.

Vector는 항상 동기화 되어있고 Collection 프레임워크에 없는 메소드들을
사용이 가능하다.

하지만 동기화라는 특징이 있어 스레드가 아닌 환경에서는 거의 사용되지 않는다.

또 항상 동기화되므로 스레드 환경에서의 안정성은 높지만 ArrayList와 비교하여
추가, 검색, 삭제의 성능은 떨어지는 단점이 있다.
