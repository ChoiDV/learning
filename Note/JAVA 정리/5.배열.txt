배열이란? 동일 자료형의 집합이다. 하나의 이름으로 여러개의 데이터를 사용할 수 있다.
여러 개의 동일자료형 변수들을 모아놓은 또 하나의 변수라고 생각하면 된다.
변수 상자들을 넣은 순서대로 번호를 매겨서(index) 큰 트럭에 싣는다고 생각하면 쉽다.
배열은 크기만큼의 공간(방)을 갖고있다. 그 공간의 번호는 0번부터 시작하며 공간(방)의 번호를 index라고 한다.
즉 ex) int[] iArr = new int[5]; 를 하면 배열의 크기는 5이고 5개의 방을 갖게되는것이다. 
배열의 인덱스는 0번,1번,2,번,3번,4번 이렇게 5개의 방을 갖게된다.
배열 선언 방법은 1. int[] iArr = {10,20,30,40,50}; // 선언과 초기화를 한번에 하는 방법
2.int[] iArr = new int[5];
iArr[0] = 10;   iArr[1] = 20; ... iArr[4]=50; // 배열선언과 배열을 생성해준뒤 각 배열방에 값을 할당하는 방법

배열은 배열을 구성하고 있는 데이터들의 주소값을 가지고 있다.
이때 주소값을 레퍼런스 라고 하고, 이런 변수들을 레퍼런스 변수라고 한다.
