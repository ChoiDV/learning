<싱글톤 패턴>
싱글톤 패턴 이란? 
어떤 클래스의 객체는 오직 하나인 유일한 객체를 만들어 여러가지 상황에서
동일한 객체에 접근하기 위해 만들어진 패턴이다. (실제로 많이 쓰이는 유형)
  
클래스 하나에 객체를 유일하게 하나만 생성하여 모든 곳에서
하나의 객체에만 접근할 수 있도록 한다.
* 즉 전역의 개념으로 객체를 사용할 수 있다. *

<스트레티지 패턴>
기능 하나를 정의하고 각각을 캡슐화하여 교환해서 사용할 수 있도록 만든다.
스트레티지를 활용하면 기능을 사용하는 클라이언트와는 독립적으로 기능을 변경할 수 있다.
어떤 객체를 만들 때 객체가 가지는 기능들이 다양하게 존재할 것이다.

이러한 기능들을 추상화 하여 언제든지 적용할 수 있게 만드는 것이다.
즉 기능을 * 부품화, 표준화 * 하는 것이다.
