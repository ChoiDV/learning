<공간분할태그>
1. 블록형식의 태그
(1) <p>...</p> 태그 
	대표적인 블록형 태그이며 단락을 나눌때 주로 사용한다. 개행이 포함된다.
(2) <div>...</div> 태그
	block형식으로 공간을 분할한다. 하나의 블록형식의 태그이므로 개행이 포함된다.
(3) <ul><li>...</li><ul> 태그
	ul안의 li태그 리스트 태그이며 이것도 개행이 포함되며 앞에 ○ 나 ● 가붙는다.

2. 라인형식의 태그
(1) span
	inline 형식으로 공간을 분할한다. 개행이 포함되지 않는다.

시멘틱을 이용한 레이아웃
HTML5 태그의 가장 큰 변화를 시멘틱 태그라고 한다.
시멘틱(semantic)은 '의미론적인' 이라는 뜻이다.
기계적인 검색 엔진은 어떠한 태그가 어떠한 기능을 하는지 분별 할수 없고 데이터를 효율적으로 추출할 수 없다.
그래서 특정 태그에 의미를 부여해서 웹 페이지를 만들기 시작했고 이를 시멘틱 웹이라고 표현한다.

HTML5의 시멘틱 구조 태그
-header : 헤더
-nav : 네비게이션
-aside : 사이드에 위치하는 공간
- section : 여러 중심 내용을 감싸는 공간
- article: 글자가 많이 들어가는 부분
- footer : 꼬리말
