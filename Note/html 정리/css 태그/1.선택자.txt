<선택자>
선택자란 HTML 문서 중 특정 부분을 선택하여, 
해당 부분의 속성을 변경하는 목적으로 사용한다.


특정 태그를 선택하여 CSS속성을 적용할 수 있다.

선택자

- tag 선택자 : 특정 태그를 선택하여 CSS속성을 적용할 수 있다.

- *  :  * 은 전체라는 의미의 선택자 이다. *선택자를 사용하면 문서 전체를 선택하는 의미가 있다.

- id(#) 와 class(.) : 태그의 id와 class를 이용하여 CSS에 속성을 설정할 수 있다.

- tag, id, class 혼합 : tag와 id, class 를 혼합하여 CSS에서 속성을 설정할 수 있다.


속성 선택자 : tag의 속성을 선택하여 CSS의 속성을 설정할 수 있다.

1- 선택자[속성] : 지정한 속성을 가지고 있는 요소를 찾아 스타일을 적용

2- 선택자[속성=값] : 속성 값이 특정값과 같은 경우를 찾아 스타일을 적용

3- 선택자[속성^=값] : 속성 값이 지정한 문자로 시작하는 속성값에 대해서만 스타일을 적용

4- 선택자[속성$=값] : 속성 값이 지정한 문자로 끝나는 속성에 대해서만 스타일을 적용



<후손 및 자손 선택자>  : tag의 속성을 선택하여 CSS의 속성을 설정할 수 있다.
ex) <div>
       <ul>
         <li>  
	<p> 태그들이 있다면 
div li  : 후손 선택자  : 밑에 있는 태그를 의미 
div p : 후손 선택자
div > ul : 자손 선택자  :  바로 밑의단에 있는 태그를 의미


<동위(=형제) 선택자 > : 태그의 동등한 위치를 판단하여 CSS속성을 설정할 수 있다.

+ : 태그의 동위 선택자들 중 바로 다음것 하나만
~ : 태그의 동위 선택자들 다 


<반응 선택자 > : 마우스를 올려 놓았을 때 (hover상태) 의 반응에 따른 속성을 설정할 수 있다.
ex)  div ul li  
li위에 마우스를 올렸을때 반응선택자를 사용하려면 

div ul li:hover {
     li위에 마우스롤 올렸을때 반응할 디자인
} 
으로 사용하면 된다.

<구조 선택자> : 구조에 따라 특정한 위치에 속하는 태그를 선택하고 속성을 설정할 수 있다.
:nth-child(수열)  : 수열에 따라 위치하는 태그 선택
:last-child : 마지막 요소를 선택
:first-child 첫 번째 요소를 선택
