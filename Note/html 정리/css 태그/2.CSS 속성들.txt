<CSS 속성들 >

--< url() : background-img 속성의 속성값으로 많이 사용된다.
이 경우 배경이미지의 경로를 나타낸다.

--< display 속성 : 화면에 어떻게 보이는지를 설정하는 속성. 다양한 속성값이 있지만,
주로 몇가지만 많이 사용된다.
1: block : div태그, p태그, li 태그 : 높이의 속성이 있다.

2: inline : span태그 : 높이의 속성이 없다.

3: inline-block : block의 속성을 갖고 옆으로 붙여 나가는 것 : 높이의 속성이 있다.

--<  visibility 속성 : display 속성의 none속성값과 비교하여 이해한다.

1: visibility:hidden은 공간은 확보, 안보임
2: display:none은 공간도 사라지고 안보인다.

--< opacity 속성 : 투명도를 조절하는 속성이다.
opacity : 0~1; 사이로 조절할수 있다.
0은 투명
1은 불투명


--< margin 및 padding 속성
margin : 밖 여백을 의미한다. : 요소와 요소의 간격을 벌려준다.

padding : 안 여백을 의미한다. : 요소안의 간격을 벌려준다.

--< box-sizing 속성 : box-sizing속성을 이용하면 전체 사이즈의 크기를 변경 할 수 있다.

--< border 속성 : border(테두리) 속성을 변경할 수 있다.
border: 테두리의 굵기 선종류 색깔; 을 설정할 수 있다.

--< 글자 관련된 속성들 
font-family(글꼴)
font-size(크기)
font-style(이텔릭체)
font-weight(진하게)
line-height(행간격 조정용도나 세로축에 대한 정렬 용도)
text-align(글자에 대한 정렬)
text-decoration(간단한 글꼴 모양의 속성)
